# mudpeter

This package can be used to download full-texts from [Pubmed](https://pubmed.ncbi.nlm.nih.gov/) and [BioC API](https://www.ncbi.nlm.nih.gov/research/bionlp/APIs/BioC-PMC/) and persist them locally in a sqlite database or optionally exported as textfiles for further analysis.

## Prerequisites

You need to have `uv` (version > 0.11.0) installed.

## Installation

Type `uv pip install git+https://github.com/joheli/mudpeter.git` to install a development version straight off this repo.

## Help

Once successfully installed, type `mudpeter --help` to get an overview of the subcommands.
